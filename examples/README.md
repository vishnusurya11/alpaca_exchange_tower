# Example Order Files

This folder contains sample order files for all supported order types.

## Usage

Copy any example file to `orders/incoming/` to test the order processor.

```bash
cp examples/paper_testbot_stockbuy_*.json orders/incoming/
```

## Order Types Included

2. **stockbuy** - paper_testbot_stockbuy_20260214074858810143.json
